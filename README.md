# Plugin-Spring

### Plugin-Spring 能做什么？

> 基于Plugin-Spring实现以下功能
- 统一输入与输出（JSON基于GSON）
- 全局错误统一
- 上下文辅助
- Cookie辅助

---
### 使用环境

    JAVA    >  1.8
    Maven   >  3.X
    Mybatis >  3.X
    
### Maven仓库
    
```xml

<dependency>
  <groupId>com.github.link-kou</groupId>
  <artifactId>spring</artifactId>
  <version>1.0.2</version>
</dependency>

```
